# 모두의 타이머 (ModoTimer) ⌛

### 타이머 실행 및 저장 앱
### 2023/10/16 기준

<주요 기능>
- 일반적인 타이머 기능
- 개인의 타이머 목록을 저장할 수 있는 기능
- 저장한 타이머 데이터를 가지고 와서 타이머 시작 기능
  
<구현 예정 기능>
- 다중 타이머 기능
- 타이머 검색 기능
- 타이머 목록 공유 기능
